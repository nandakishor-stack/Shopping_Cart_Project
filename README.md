# 🛒 Shopping Cart Web Application

A modern and responsive Shopping Cart application built using **React.js** and **Tailwind CSS**. This project allows users to browse products, add items to the cart, update quantities, and view the total price dynamically.

---

## 🚀 Features

- 🛍️ Add products to cart
- ❌ Remove items from cart
- 🔢 Update item quantity
- 💰 Real-time total price calculation
- 💾 Persistent cart using Local Storage
- ⚡ Fast and dynamic UI with React
- 🎨 Clean and responsive design using Tailwind CSS

---

## 🧑‍💻 Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **State Management:** useState / useEffect
- **Storage:** Local Storage

## 🧠 How It Works

- Products are rendered dynamically using React components
- When user clicks **Add to Cart**:
  - Item is stored in state
  - Synced with Local Storage
- Cart updates instantly when:
  - Quantity changes
  - Item is removed
- Total price is calculated using state updates
